`BotLogs` houses repositories associated with [BotLogs.dev](https://botlogs.dev) — the AI‑powered error monitoring platform for Discord bots.

Stop wasting time debugging your bot. BotLogs automatically captures errors, analyzes them with AI, explains the issue simply, and can even create a pull request on GitHub.

To get started, grab your API key at [botlogs.dev](https://botlogs.dev) and follow our [quickstart guide](https://docs.botlogs.dev).
